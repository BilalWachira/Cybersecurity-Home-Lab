# Home Lab Objectives

## Primary Objective

Build a safe enterprise-style environment where I can practise cybersecurity, Windows administration, networking and GRC concepts.

## Technical Objectives

- Install and manage Windows Server 2022
- Configure Active Directory
- Configure DNS and DHCP
- Create and manage users and groups
- Apply Group Policy
- Join Windows workstations to a domain
- Learn Windows and Linux administration
- Practise networking and troubleshooting
- Introduce security monitoring tools

## GRC Objectives

- Understand security controls in practice
- Identify risks associated with enterprise systems
- Document assets and configurations
- Produce audit-ready evidence
- Connect technical configurations to security policies
- Understand identity and access management
- Practise change and configuration management

## Career Objective

Use the lab to develop practical evidence for entry-level cybersecurity and GRC roles while building toward long-term information-security leadership.
